🔧 V1.1 — 安全与稳定性修复

1.1 修复 Scanner 种子入队阻塞死锁 P0

问题

scanner/scanner.go:80 — 当 len(seeds) > QueueSize 时，同步投递 s.taskCh <- task 阻塞整条链路，worker goroutine 无法调度启动。

修复

将种子投递移入独立 goroutine 异步执行，确保 worker 已启动后再投递：

- for _, task := range seeds {
-     s.taskCh <- task  // 阻塞！
- }
+ go func() {
+     for _, task := range seeds {
+         s.taskCh <- task
+     }
+ }()

影响文件: internal/scanner/scanner.go

1.2 修复 Context 取消时 pending 泄漏 P0

问题

scanner/scanner.go:136-138 — worker 收到 ctx.Done() 后直接 return，未调用 pending.Done()。已入队但未处理的 task 永远不被消费，pending.Wait() 永久挂起，Wait() 死锁。

修复

新增 drainPending() 方法，ctx 取消时非阻塞地清空 taskCh 并逐一调用 pending.Done()：

+ func (s *Scanner) drainPending() {
+     for {
+         select {
+         case <-s.taskCh:
+             s.pending.Done()
+         default:
+             return
+         }
+     }
+ }

影响文件: internal/scanner/scanner.go

1.3 OpenList Token 自动刷新 P1

问题

Token 仅在启动时通过 Login() 设置一次，无刷新机制。长期扫描 (数小时) 后 token 过期导致 401，ErrAuth 不可重试，直接失败。

修复
  1. Client 结构体增加 sync.Mutex 保护 token 并发读写
  2. request() 检测到 401 + username 已设置时，自动调用 reLogin()
  3. reLogin() 仅重试一次，避免无限循环
  4. doRequest() 读取 token 时加锁

影响文件: internal/openlist/client.go

1.4 配置文件安全 P1

问题

configs/config.yaml 明文提交了 password: "admin" 和 TMDB API Key，存在安全隐患。

修复
  • 从 YAML 中移除 password 和 api_key 明文值
  • 添加注释说明环境变量覆盖方式
  • 创建 .env.example 参考文件（已被 .gitignore 忽略）

影响文件: configs/config.yaml， 新建 .env.example

问题

2.1 年份识别修正 P2

问题

文件名 2012.mkv 会将 2012 同时作为标题和年份，可能导致 TMDB 错误匹配。

修复

在 Normalize() 末尾增加检查：当 Title == Year 字符串时（纯数字文件名），将 Year 抑制为 0。TMDB 搜索仅用名称仍然可以正确匹配。

+ if info.Year > 0 && cleaned == itoa(info.Year) {
+     info.Year = 0
+ }

2.3 媒体质量指纹 P2

问题

去重算法仅依靠文件大小聚类，不区分 4K REMUX / 1080p WEB 等质量差异。两个不同质量的同部电影可能被误判为重复。

修复

新建 media/fingerprint.go，实现：

  • MediaFingerprint 结构体：Resolution / Codec / HDR / Audio / Source
  • ExtraceFingerprint(name) — 从文件名提取质量属性
  • Score() — 返回量化品质评分 (分辨率 ×1000 + 编码/HDR/音质/来源加成)
属性识别内容加分范围
分辨率SD/HD/FHD/4K/8K1000–5000
编码AV1/HEVC/x265 → x264/AVC100–300
HDRDV → HDR10+ → HDR10 → HLG150–400
音质Atmos → TrueHD → DTS-HD → DTS → AAC120–500
来源REMUX → BluRay → WEB-DL → WEBRip30–500

影响文件: 新建 internal/media/fingerprint.go

问题

storagePriority 硬编码为 local > tianyi > quark，影视场景应质量优先。

+ fp := media.ExtraceFingerprint(f.Name)

影响文件: internal/duplicate/duplicate.go
